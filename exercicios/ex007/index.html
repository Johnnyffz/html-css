Links Externos

Nota

Para colocar um link na pagina, basta colocar a Tag "a", para abrir em outra guia a tag "target = _blank", e utilizar "rel=" para indicar o que é aquele link

EX:

Esse é o meu site oficial, para acessar o nosso conteudo 100% gratuito Clique Aqui
--- <a href="https://link..." target="_blank" rel="external">Clique Aqui</a>
