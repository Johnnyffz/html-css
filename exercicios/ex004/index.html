Outras formatações

Tag Code / Pre

Para inserir um codigo em sua criação em html, voce pode utilizar a tag code para deixá-lo em uma formatação mais apresentável, ex: num = int(input('Digite um número: ')) num1 = int(input('Digite outro número: ')) s = num+num1 print('A soma entre {} e {} é igual à {}'.format(num,num1,s))

Percebeu algo de errado nessa formatação? SIM! A formatação em si não está errada, mas sim a forma como o codigo está escrito. Um código sem quebra de linha fica muito bagunçado e feio, para resolver isso basta dar um simples comando <pre>, ex: 

num = int(input('Digite um número: '))
num1 = int(input('Digite outro número: '))
s = num + num1
print('A soma entre {} e {} é igual à {}'.format(num,num1,s))

Citações Simples

Para fazer uma referencia a uma fala ou uma citação de alguem, utiliza-se a tag <q>, ex: Creio no riso e nas lagrimas como antídotos contra o ódio e o terror - Charlie Chaplin

Citações completas

Para fazer citações completas, basta utilizar a tag <blockquote>, como por ex:

Segundo Dalai Lama:

Se você quer transformar o mundo, experimente primeiro promover o seu aperfeiçoamento pessoal e realizar inovações no seu próprio interior. Estas atitudes se refletirão em mudanças positivas no seu ambiente familiar. Deste ponto em diante, as mudanças se expandirão em proporções cada vez maiores. Tudo o que fazemos produz efeito, causa algum impacto.

Abreviações

Para abreviações como HTML ou CSS, Utiliza-se a tag <abbr>.
