HTML E CSS são Linguagem de programação?

Muitas pessoas acham que HTML e CSS são linguagens de programação, mas será mesmo que essa teoria esta correta? 🤔

Pois bem, em tese a teoria não está totalmente equivocada, visto que, a HTML e CSS são sim uma linguagem, mas não de programação, e sim de marcação que tem como objetivo marcar e estruturar o site de forma gráfica e através de códigos que organizam a página através de comandos.

Um exemplo de linguagem de programação é o JavaScript, que ai sim tem o objetivo de programação e interatividade com a pagina do site.

Exemplo de código JS

        
    const para = document.querySelector("p");

    para.addEventListener("click", atualizarNome);

    function atualizarNome() {
    var nome = prompt("Insira um novo nome");
    para.textContent = "Jogador 1: " + nome;
    }
