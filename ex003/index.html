Principais formatações

Negrito / Destaque

Podemos ver aqui o exemplo de fonte em NEGRITO, utilizando o codigo <strong>

Itálico / Ênfase

Para dar ênfase podemos utilizar o codigo <em>

Marcação

Para marcar um texto assim, basta utilizar a tag <mark>.

Ou para adicionar uma cor diferente, inserir a tag 'style' dentro de <mark>

Texto grande / pequeno

Estamo criando um texto grande e um texto pequen. (Tag <big> e <small>)

Texto deletado / innserido

Exemplo de texto deletado utilizando a tag <del>

Podemos sublinhar um texto ou mostrar que ele foi inserido, utilizando a tag <ins>

Texto subrescrito / subescrito

Para um texto sobrescrito tipo, xy2 utiliza-se a tag <sup>

Para escrever um texto sobrescrito tipo, H2O é utilizada a tag <sub>
